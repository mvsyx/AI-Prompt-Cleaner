# AI Prompt Cleaner

A lightweight Python tool for cleaning and structuring prompts for Large Language Models (LLMs).

## Features

- Prompt normalization
- Prompt structuring
- Markdown export
- Command-line interface

## Example

Input:

write me a blog post about artificial intelligence

Output:

Role: Professional Content Writer

Task: Write a blog post about artificial intelligence.

Requirements:
- Clear structure
- Professional tone
- Practical examples
